# Modern Literacy
Modern Literacy builds AIDEN, a multi-repo AI governance workbench focused on policy-backed review, evidence, and controlled deployment paths.

## Repository tour
- `aiden` — hub repo for architecture, orchestration, and the top-level story
- `aiden-engine` — core evaluation engine and shared contracts
- `aiden-demo` — interactive demo surface
- `aiden-api` — API adapter layer
- `aiden-tools` — evaluation harnesses and supporting tooling
- `aiden-policies-*` — versioned policy packs for domain-specific review

## Working norms
- Changes to the default branch flow through pull requests.
- Repository ownership is enforced through `CODEOWNERS`.
- Security-sensitive concerns should follow the reporting path in `SECURITY.md`.
